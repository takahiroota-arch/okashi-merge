# おかしまーじ！ イラスト仕様書（assets/）

このフォルダに下記ファイル名でPNG（透過）を入れて push すると、ゲームが自動で
イラストに切り替わります。**ファイルが無い場合は絵文字/手描きにフォールバック**するので、
1枚ずつ差し替えてもOK。共通ルール：

- 形式：PNG・背景透過
- トンマナ：パステル＋元気・可愛い（丸ゴシック「おかしまーじ！」の世界観）
- キャラ（メイドさん）は全表情で**同一人物・同一構図（バストアップ）・同じ目線**にすると差し替えがきれい
