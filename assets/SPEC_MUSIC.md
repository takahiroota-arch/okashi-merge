# おかしまーじ！ 楽曲(BGM)仕様書

外部ツールで作曲したら、`assets/` に下記ファイル名で置いて push すると**自動で再生**されます。
ファイルが無い場合は無音（または簡易フィーバー音）にフォールバックするので、1曲ずつでもOK。

## 共通
- 形式：**MP3**（.mp3）。広く再生できるのでMP3推奨（OGGでも可だが要相談）
- ループ素材は**シームレス**に（先頭と末尾が繋がる／端に無音・プツ音を入れない）
- 音量は作り込みすぎなくてOK（ゲーム側で抑えめに鳴らします）
- ブラウザ仕様で**最初のタップ後**に鳴り始めます（自動再生制限）

---

## 1. `bgm.mp3` （必須）— メインBGM
| 項目 | 内容 |
|---|---|
| 入れどころ | カウントダウン後のプレイ開始 〜 ゲームオーバーまで、ずっとループ |
| 長さ | 30〜60秒のループ |
| テンポ | BPM 120〜140 |
| 雰囲気 | 可愛い・元気・パステル。お菓子作りカフェ風。明るく軽快 |
| 楽器例 | トイピアノ／マリンバ／グロッケン／鈴／軽いポップなビート |
| 音量目安 | ゲーム側 0.30 で再生（控えめ）|

## 2. `bgm_fever.mp3` （任意）— フィーバー中の曲
| 項目 | 内容 |
|---|---|
| 入れどころ | フィーバー発動中（5秒）だけ。終わるとメインBGMに自動で戻る |
| 長さ | 5秒以上のループ（短ループでOK）|
| 雰囲気 | メインを一段アップテンポ＆派手にした版。最高潮の盛り上がり |
| 音量目安 | ゲーム側 0.38 |
| 無い場合 | メインBGMのまま、簡易フィーバー音で代替（実装済み）|

## 効果音(SE)について
SEはゲーム内で自動生成しています（外部不要）。今回**サイン/三角波＋ローパス＋エコー**で
柔らかく可愛い音に調整済み。「もっと丸く／キラキラ」等の要望があれば数値で再調整できます。
